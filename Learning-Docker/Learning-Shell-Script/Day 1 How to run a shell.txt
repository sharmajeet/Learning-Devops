*How to run a shell script program in linux machine*
-commands-
1 -> vim hello.sh
2 -> in vim i for insert , :wq for save and exit , :qa for exit without save 
3 -> ls -l hello.sh , for checking permissions given to file
4 -> give permission to file 'Read-Write-Execute' 000/111 - 7/5/4/2/1
5 -> now execute that file ./hello.sh
